# Brew Shift Core Prototype
This version of the project has been rebuilt into a simple working Flutter prototype using Firebase.

## Core functionality included
- role-based login for employee and manager accounts
- local user registration
- employee clock in
- employee clock out
- automatic daily timesheet creation/update
- employee daily timesheet view
- employee upcoming rota view
- manager view of today's timesheets
- manager rota creation
- manager view of upcoming rota entries

## Quick local run steps
1. Open the project folder in VS Code or Android Studio.
2. Run `flutter pub get`.
3. Start an emulator or connected device.
4. Run `flutter run`.
5. Test login, register, clock in/out, timesheet display and rota creation.
